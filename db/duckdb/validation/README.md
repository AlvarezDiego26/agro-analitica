# Validacion DuckDB

Aqui van las consultas de prueba para confirmar que DuckDB:

- carga las extensiones necesarias
- puede autenticarse contra MinIO
- puede leer tablas Delta reales
- responde correctamente con filtros y conteos basicos

Esta carpeta es util para levantar el stack en otra maquina y validar rapido si el entorno quedo funcional.
